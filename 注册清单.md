## ① GitHub（唯一需要注册的）

1. 打开 https://github.com/signup
2. 填邮箱 → 设密码 → 设用户名（随便起，比如 `xiaonan`）→ 验证人机 → 点 **Create account**
3. 去邮箱查验证邮件，点里面的确认链接
4. 登录后，右上角 **➕** → **New repository**
   - Repository name 填：`strawberry-family`
   - 选 **Public**
   - 不要勾任何初始化选项 → 点 **Create repository**
5. 然后二选一（推荐 A）：
   - **A（我帮你推代码）**：点自己头像 → **Settings** → 左侧 **Developer settings** → **Personal access tokens** → **Tokens (classic)** → **Generate new token (classic)** → Note 随便写、Expiration 选 No expiration、勾上 **repo** 那一项 → **Generate token** → 复制那一长串 `ghp_开头` 的字符
   - **B（自己传）**：在仓库页面把整个 `小草莓家族-cloud` 文件夹拖拽上传（不用 token）

✅ 注册完把这两样发我：
- 仓库地址：`https://github.com/你的用户名/strawberry-family`
- （选 A 的话）那个 `ghp_...` token

## ② Supabase（存数据，用 GitHub 登录）

1. 打开 https://supabase.com → 点 **Start your project** → 选 **Continue with GitHub** → 授权
2. 登录后 **New project**
   - Name：`strawberry-family`
   - Database Password：点生成那个眼睛图标复制保存（不重要，记不住也没事）
   - Region：选 **Northeast Asia (Tokyo)**
   - 点 **Create new project**（等 1~2 分钟）
3. 进去后左侧 **SQL Editor** → **New query** → 把 `supabase-schema.sql` 文件里的全部内容粘进去 → 点 **Run**
4. 左上角点项目名 → **Project Settings** → **API**
   - 复制 **Project URL**（形如 `https://xxxx.supabase.co`）
   - 复制 **anon public** 那一栏的 key（一长串 `eyJ...`）

✅ 注册完把这两样发我：
- Project URL
- anon key
